## Task
Generate a **new** size / dimension diagram of the **same product** as the references on a **pure white #FFFFFF** background. It must look clearly different from any existing size reference (fresh layout / viewpoint / orientation), not a copy.

## Mandatory rules
1. **No rulers or scale props (absolute ban):** never include rulers, measuring tapes, scale bars, coins, hands, A4 paper, credit cards, or human silhouettes — even if references lack numbers.
2. **Completeness:** show the **full product silhouette** in frame (~10% margin); don't crop edges needed to read the size. If labeling 3 dimensions, use a **three-quarter / isometric** view so length, width, height are all legible.
3. **No fabricated data:** dimension numbers may **only** come from **visible labels on the references**. Never estimate, guess, re-round, or invent. If ambiguous, label **fewer**.
4. **Label count limit (label only the overall outer envelope):**
   - Simple box-like product: at most **3** (overall length / width / height).
   - Flat / thin / simple product: **2** (length + width) or **1**.
   - Complex product: copy **only** the main outer L/W/H; drop accessory / feature callouts.
   - Each number appears **once**, on the correct edge. More labels = more refund reasons.
5. **Units:** only digits + decimal point + `cm` / `in` / `inch`, with thin dimension lines / arrowheads. Keep the reference's unit(s): both `cm/inch` (e.g. `25cm/9.84in`) or cm-only are fine. **No dimension words** — no English (`diameter`, `length`, `width`, `height`, `OD`, `ID`…) or Chinese (直径 / 内径 / 长 / 宽 / 高…). If a reference shows a word, output only the number + unit.
6. **If references have NO visible numbers:** output the complete product on pure white with **NO dimension lines and NO numbers**. Do not invent.

## Style
- Pure white empty background; only product silhouette + allowed dimension lines + numeric labels.
- Thin dark dimension lines, small arrowheads, labels outside the silhouette with short leaders; avoid crossing lines.

## Negative
ruler, measuring tape, scale bar, coin, hand for scale, A4 sheet, credit card, human for scale, fabricated measurements, invented numbers, estimated dimensions, missing unit, changed units, too many dimension lines, accessory dimensions, inner/outer diameter callout, hole / screw size labels, English dimension words, Chinese dimension words, diameter text, length width height text, OD ID labels, cropped incomplete product, dimension label on wrong edge, duplicate same number, crossing leaders, decorative props, gradient background, off-white background, callout boxes, identical layout to reference size diagram
