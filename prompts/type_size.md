## Task
Use the provided reference image as the source dimension diagram. Do not create a new product diagram from imagination. Do not choose another reference. Edit only this selected dimension image.

## Absolute preservation rules
1. Keep the product itself unchanged: same product, same shape, same color, same angle, same proportions, same visible parts.
2. Keep every visible dimension number unchanged: same digits, same decimals, same unit, same value, same meaning.
3. Do not invent, estimate, calculate, round, translate, convert, add, remove, or correct any dimension data.
4. If a dimension value is unclear, preserve it as shown; do not replace it with a guessed value.
5. Do not add new measurements based on product title, product appearance, common sense, or assumptions.

## Allowed edits only
- Restyle dimension lines: cleaner thin lines, consistent stroke width, neat arrowheads, improved alignment.
- Restyle dimension text: clearer font, better contrast, cleaner placement, consistent size.
- Improve spacing and readability of existing labels without changing the product or the numeric values.
- Clean minor visual noise around the labels and dimension lines.
- Keep the background clean, preferably pure white or the same white style as the source diagram.

## Label rules
- Numeric dimension labels must remain exactly as in the source image.
- Units must remain exactly as in the source image (`cm`, `mm`, `in`, `inch`, etc.).
- Do not add words like length, width, height, diameter, OD, ID, or Chinese dimension words unless they already exist and must be preserved.
- If the source has only numbers and units, output only numbers and units.

## Style
- Professional Amazon/e-commerce dimension image.
- Clean, readable, minimal, technical, accurate.
- Product and original measurement data are the authority.
